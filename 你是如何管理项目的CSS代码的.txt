在 HTML + CSS + JS 的项目中，不使用行内样式；推荐使用单独的 CSS 文件编写样式，然后在 HTML 的 head 标签内，使用 link 引入；如果 CSS 不需要在多个页面共享，可以通过 style 标签写在文件内部，同样推荐放置在 head 标签内。CSS 隔离于 HTML 单独编辑方便后期的维护，放在 head 中可以确保样式相比于页面内容提前加载，展现内容不会出现样式错乱。
